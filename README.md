# Bluetooth Controlled Robo Race Car

A smartphone-controlled robotic vehicle built using Arduino Uno, HC-05 Bluetooth Module, and L298N Motor Driver.

The robot communicates wirelessly with a mobile device through Bluetooth and executes real-time motion commands. The project was developed for a Robo Race competition and includes speed control, directional movement, diagonal movement, and electronic braking functionality.

## Features

- Forward / Reverse Motion
- Left / Right Steering
- Diagonal Movement
- Variable Speed Control
- Electronic Braking System
- Wireless Bluetooth Communication

## Hardware Used

- Arduino Uno
- HC-05 Bluetooth Module
- L298N Motor Driver
- 2 DC Motors
- Robot Chassis
- Battery Pack

## Software Used

- Arduino IDE
- Embedded C
- Bluetooth Controller Mobile App
